# Maize Bag CO2 and Moisture Experiment Protocol

## Purpose
Measure CO2, humidity, and temperature inside sealed plastic bags with dried maize to compare moisture retention across materials.

## Materials
- PP, PVC, Nylon bags (large enough to hold maize and sensors)
- Dried maize (corn)
- CO2 sensor
- Humidity/moisture sensor
- Temperature sensor
- Moisture probe for grain

## Setup
1. Place dried maize and all three sensors inside each bag
2. Seal bags
3. Place one corn sample outside as control
4. Record starting readings (day 0)

## Timeline
| Day | Action |
|-----|--------|
| 0 | Seal bags, record baseline |
| 1-2 | Monitor daily; log CO2, humidity, temperature |
| 3 | Open bags to prevent CO2 sensor saturation; take moisture probe reading |
| 4-5 | Continue monitoring open bags |

## Data recording
- Log CO2 (ppm), humidity (%), temperature (C) at each reading
- Take average of multiple sensor readings to reduce noise
- Record control (outside bag) at each point

## Notes
- John Deere moisture tubes were unavailable; use available moisture probe
- CO2 sensor limit reached on day 3 for PP and Nylon bags
